Heroku-Mondrian API

Notes

Resource URL: /api/width/height/levels/wRatio/hRatio/discardRatio.ext

  • Non-conforming values are ignored and replaced with defaults
  • Would recommend using a 2x size for retina displays

Parameters

Argument Default Description Required Example
width 200px Image width; integer > 0 Optional 200
height 300px Image height; integer > 0 Optional 200
levels 3 Recursion depth. 2x number of squares. Integer > 0. For best results, < 10. Optional 3
wRatio 0.45 Ensures vertical cuts don't make squares too thin. Float, 0 > x > 1 Optional 0.45
hRatio 0.45 Ensures horizontal cuts don't make squares too short. Float, 0 > x > 1 Optional 0.45
discardRatio true Whether to discard squares that are too thin or too short. Optional {true,false}
ext .png Extension to receive. Must be one of `jpg`, `webp`, or `png`. Optional {png,jpg,webp}
